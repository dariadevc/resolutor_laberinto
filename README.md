# Recorredor de caminos

Simulación de una Máquina de Turing. Permite mostrar el recorrido que se obtiene a partir de la misma.

Desde la terminal se pueden observar tanto la cadena de entrada como la de salida:

- Cadena de entrada: Es el camino, separando las filas con # de por medio. Los 0 representan el camino y los 1 las paredes.
- Cadena de salida: Es el camino, en lugar de 0 hay letras que indican la dirección que hay que seguir para llegar a la salida: A = Abajo, D = Derecha e I = Izquierda

## Dependencias

Se debe instalar la librería pygame (https://www.pygame.org/news) con el siguiente comando desde la terminal:
`pip install pygame`

## Versiones

- python 3.10.12
- pygame 2.6.1
